## Input  

User's input box, enter some content for submission to the server.

## Usage

```js
<o-input  placeholder="Enter your name" />
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------:|
| placeholder  | string|       |     占位字符串      |
