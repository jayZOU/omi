## Toast 

Simple reminder notification.

## Usage

```js
<o-toast type='success' show={true}>支付成功</o-toast>
<o-toast type='loading' show={false}>加载中</o-toast>
```

## API

### Props

|  **Name**  | **Type**        | **Defaults**  | **Details**  |
| ------------- |:-------------:|:-----:|:-------------:|
| show  | bool | false  |  |
| type  | string | -- |Options: loading, success|
